# Ralph Progress Log
Started: Tue 25 Mar 2026 - Visual Redesign

## Codebase Patterns
- Use `source venv/bin/activate && mypy robot_arm.py` to typecheck (venv has mypy and pygame stubs)
- All drawing functions are in `robot_arm.py` — no separate modules
- Arm segments use rotated polygon corners pattern: define corners relative to pivot, rotate with cos/sin, draw with `pygame.draw.polygon`
- Color constants are defined at module level as `tuple[int, int, int]`
- `draw_arm1` returns end point `(float, float)` that chains into `draw_arm2`
- Base uses stepped shape: wider platform bottom + narrower pedestal top. Pivot is at `GROUND_Y - BASE_HEIGHT` (top of pedestal).
- Shadow/3D effect pattern: draw a darker offset rect behind the main rect (offset ~3px)
- Tapered arm pattern: use wider half-width at pivot end and narrower at tip for trapezoidal polygon
- Use a local `rotate(cx, cy)` helper inside draw functions to keep rotation code DRY
